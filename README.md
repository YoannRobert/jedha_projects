# Projets de certification CDSD

Dépôt regroupant les huit projets réalisés dans le cadre de la certification "Concepteur Développeur
en Science des Données" (Jedha), répartis en cinq blocs. Chaque projet dispose de son propre README
détaillé dans son dossier.

| Bloc | Projet                                                                                                           | Compétences clés              | Stack technique                                   |
|------|------------------------------------------------------------------------------------------------------------------|-------------------------------|---------------------------------------------------|
| 1    | [Kayak](bloc1/Kayak) - pipeline de recommandation de destinations et d'hôtels                                    | Data engineering (ETL)        | Scrapy · AWS S3 · AWS RDS PostgreSQL · SQLAlchemy |
| 2    | [Steam](bloc2/Steam) - facteurs du succès d'un jeu vidéo                                                         | EDA (big data)                | PySpark · Databricks                              |
| 2    | [Tinder](bloc2/Tinder) - ce qui motive un second rendez-vous                                                     | EDA                           | pandas · Plotly · statsmodels                     |
| 3    | [Conversion Rate Challenge](bloc3/Conversion_Rate_Challenge) - prédiction d'abonnement à une newsletter          | ML supervisé (classification) | scikit-learn · XGBoost · Plotly                   |
| 3    | [Uber Pickups](bloc3/Uber_Pickups) - hot zones de demande à New York                                             | ML non supervisé (clustering) | scikit-learn · GeoPandas · Plotly                 |
| 3    | [Walmart Sales](bloc3/Walmart_Sales) - prédiction du chiffre d'affaires hebdomadaire                             | ML supervisé (régression)     | scikit-learn · pandas · Plotly                    |
| 4    | [AT&T Spam Detector](bloc4/ATT_Spam_Detector) - classification de SMS spam vs ham                                | Deep learning (NLP)           | PyTorch · Hugging Face Transformers · tiktoken    |
| 5    | [Getaround Analysis](bloc5/Getaround_Analysis) - seuil entre locations et prédiction du prix journalier          | Industrialisation ML (MLOps)  | FastAPI · MLflow · Docker · Hugging Face Spaces   |
| 6    | [Projet RTE](https://github.com/YoannRobert/DDay_FullStack) - prédiction de la consommation électrique en France | Management de projet Data     | Prophet · AWS S3 · AWS EC2 · Streamlit            |
